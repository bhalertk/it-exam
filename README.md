# 115 資訊管理實務 考試重點

華南商業銀行內部考試「115\_資訊管理實務」的線上複習工具，涵蓋五大部門命題範圍的重點整理與練習題庫。

## 🔗 線上使用

👉 **[開始複習](https://<YOUR_USERNAME>.github.io/it-exam/)**

> 純前端靜態頁面，無需安裝，手機 / 平板 / 電腦皆可使用。

## 📖 功能

| 功能 | 說明 |
|------|------|
| **閱讀模式** | 五大部門考試重點，側欄導航 + 關鍵字搜尋 |
| **測驗模式** | 142 題選擇題，可依部門篩選，每次隨機 20 題 |
| **深色 / 淺色主題** | 右上角切換，自動記憶偏好 |

## 📚 命題範圍

| 部門 | 題數 | 主要內容 |
|------|------|---------|
| 行政管理部 | 15 | 資訊大樓宿舍管理及使用須知 |
| 人力資源部 | 23 | 工作規則第17條、員工平時考核要點、重要函文 |
| 資訊安全部 | 42 | ISMS、資安政策、教育訓練、防火牆、防毒、雲端服務、弱點修補 |
| 資訊作業服務部 | 37 | 電腦系統分類、密碼設定、最高權限、機房管理、變更管理、監控巡檢 |
| 資訊規劃開發部 | 25 | 應用程式管理、變更管理、外包人員管理、軟體品質、使用者權限 |
