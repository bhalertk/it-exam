## 壹、核心系統管理業務（第二章）

### 一、主機系統（第一節）

**機房管理重點：**
- 機房採**封閉式**，電腦卡管制進出
- 錄影設備影像資料儲存至少**三個月**備查
- 可進出人員：資訊作業服務部**主管、副主管**及經主管**專案核准**人員
- 臨時進入須於「資訊管理群電腦機房進出登記簿」登記

**核心系統架構：**
- 大型主機核心系統：**56項**主要核心業務（存放款、匯兌、外匯、ATM、網銀等）
- 財金跨行閘道系統（**ITGK**）：ATM及通匯跨行交易
- 跨平台資料交換系統（**C:D**）：主機與伺服器檔案傳輸中介
- 主機架構：**雙主機平行處理**（Parallel Sysplex），同地+異地備援

**硬體配置：**
- 台北主中心：**2台**主機、1台平行處理串聯器、**2座**磁碟機
- 中壢第二資訊中心：**1台**備援主機、**2座**備援磁碟機

### 二、資料庫系統（第二節）⭐
- **IMS**：連線交易系統，**階層式**資料庫
- **DB2**：**關聯式**資料庫
- 效能指標：I/O 回應時間（Response time）

### 三、備援演練（第四節）⭐⭐
- 每年應至少**一次**關鍵資訊作業演練及檢討
- **同地備援**：雙主機平行處理、磁碟機高速切換（**HyperSwap**）、GDPS/PPRC
- **異地備援**：切換至中壢第二資訊中心

## 參、資訊內控管理（第四章）⭐⭐

### 一、帳號存取控管
- 集中控管：使用 **IT維運權限管理系統**
- 作業系統最高權限帳號：Unix的**root**、Windows的**administrator**
- 使用者閒置超過 **15分鐘**，應強制自動登出
- 程式維護人員營業日上班前由RACF **REVOKE**（回收權限），需另於SBM申請

### 二、金融卡廢卡銷毀（SR-020）⭐
- 作廢之晶片金融卡由**資訊服務科**打洞作廢
- 由**資訊內控管理科**專人保管
- 每隔**二年**或累計**二千張**時，簽報**副總經理**核准，由**駐行稽核**會同銷毀

### 三、最高權限管理（SR-032）⭐⭐
- 密碼分為 **A/B 兩部分**，分別密封置於機房保險櫃
- 保險櫃鑰匙一式兩份：一份由**資訊內控管理科主管及副主管**分持，另一份存機房專櫃
- 網域特權帳號密碼每 **三個月**變更一次
- 正式環境使用時間：營業日**下午一時**起至翌日零時止
- 操作行為應**錄影留存**稽核軌跡，備份至少**五年**
- 每一營業日抽檢 **10件**操作錄影
- 營業時間外須使用密碼函，次一營業日應**重新設定密碼並重新彌封**

## 肆、連線管理業務（第五章）⭐

### 一、IP管理
- 採**白名單**方式，IP須與設備**MAC address**綁定
- 每年至少**一次** IP 位址清查

### 二、端末設備報表
| 報表 | 說明 | 決行人員 |
|------|------|---------|
| **LD01** | 外接儲存裝置使用寫入紀錄報表 | **單位副主管**以上 |
| **LD02** | 共用資料夾使用紀錄表（週報） | 線上簽核 |

- LD01連續**兩個月超過三筆**未簽核→**發文扣分**

### 三、端末電腦特殊權限清查（第三節）⭐
- 每**半年**清查一次，項目：最高權限、加入網域、安裝資產管理系統、共享資料夾、USB讀寫、燒錄機寫入
- **物聯網設備**管理清冊至少**每年更新**一次

### 四、ATM管理（第四節）
- 監控系統：**DMS**（派版、監控）、**UHD**（基本資料管理）
- ATM派版流程：資規部提出→資安部掃描→白名單解開→DMS排程→科主管覆核→全行派版

### 五、硬碟銷毀⭐
- 報廢硬碟裝入專用密封袋，送回資訊作業服務部
- 處理方式：**消磁→物理破壞→委由合格環保廠商實體銷毀**

## 伍、密碼設定原則（SR-029）⭐⭐

| 項目 | 規定 |
|------|------|
| 密碼長度 | 至少 **12個字元**（系統不支援者最少**8個字元**） |
| 變更週期 | 最長**90天**或**13週** |
| 歷程紀錄 | 至少前**3代**不重複 |
| 錯誤鎖定 | 連續**5次**輸入錯誤→帳號停止使用 |
| 最短使用期限 | 至少**5天** |
| 物聯網設備密碼 | 至少**6位**（SR-060） |

## 陸、電腦系統分類（SR-003）⭐⭐

| 類別 | 定義 | 舉例 |
|------|------|------|
| **第一類** | 直接提供客戶自動化服務或對營運有**重大影響** | 電子銀行、分行櫃台、ATM、SWIFT |
| **第二類** | 經**人工介入**直接或間接提供客戶服務 | 作業中心、客戶服務 |
| **第三類** | 未接觸客戶資訊或服務且對營運**無影響** | 人資、財會、總務、物聯網設備 |

- **關鍵資訊作業** = 屬第一類且對營運有重大影響之電腦系統
- 營運衝擊分析（BIA）：由資訊作業服務部每**半年**執行一次

### 重要名詞 ⭐

| 縮寫 | 全名 | 說明 |
|------|------|------|
| **MTPD** | Maximum Tolerable Period of Disruption | 最大可容忍中斷時間 |
| **RTO** | Recovery Time Objective | 復原時間目標 |
| **RPO** | Recovery Point Objective | 資料回復時點目標 |
| **BIA** | Business Impact Analysis | 營運衝擊分析 |

## 柒、緊急事故處置（SR-011）⭐⭐

### 緊急事故處置小組
- 由資規部、資作部、資安部組成
- **召集人**：資訊作業緊急事故發生部門**部主管**

| 小組 | 負責單位 |
|------|---------|
| 事故通報組 | 資訊作業服務部**副主管及資訊服務科**主管 |
| 軟硬體技術組 | 資作部副主管、資安部副主管及相關科主管 |
| 應用系統組 | 資訊規劃開發部**副主管及各科主管** |

### 通報程序
1. 各單位→向**資訊服務科**通報
2. 資訊服務科於**30分鐘內**→通報群主管、相關主管、業管單位、受影響單位
3. 影響客戶權益→立即通知**客戶服務部**

## 捌、機房管理須知（SR-037）⭐

| 條文 | 重點規定 |
|------|---------|
| 第3條 | 錄影保存至少**三個月**；攜入可攜式電腦/儲存媒體須**確定無病毒** |
| 第8條 | 不得攜入**磁性、放射性、易燃性、易爆性**物品；嚴禁嬉戲、吸煙、飲食 |
| 第10條 | 機房設備非經主管核准，**不得任意移動** |
| 第14條 | 下班時間後登入主機系統須**事前報請主管核准** |
| 第17條 | 溫度**16~25°C**；相對濕度**40%~70%** |
| 第19條 | 滅火藥劑以**對設備損害最小**為原則（避免灑水式） |
| 第25條 | 送修設備應將**機密等級資料移除**；無法移除者依廠商**保密切結**辦理 |
| 第26條 | 報廢前**確實清除限閱等級以上資料**；交由合格廠商銷毀，資訊部門**派員監督並拍照** |

## 玖、伺服器管理（SR-026）

- 系統日誌備份保存至少**五年**
- 伺服器管理人員每日**監看**系統日誌警告/錯誤訊息
- 內控管理人員每日監看**異常登入失敗紀錄**
- 上線前應**關閉不必要服務**、執行弱點掃描及修補

## 拾、變更管理（SR-013）⭐

### 變更等級

| 等級 | 情形 |
|------|------|
| **一般變更** | 不致產生重大影響、例行作業、可迅速回復 |
| **重要變更** | 產生重大影響、重要資源可能毀損、多數單位受影響、無法回復、核心系統架構調整或跨版本升級 |

### 農曆春節變更管制（第29條）⭐
- **假期始日前一個月**起至假期結束，原則**禁止**營運環境變更
- 例外：資安、稽核缺失改善、法令遵循、主管機關要求、涉及客戶權益、異常修正
- 例外變更須經**業管單位群群主管及資訊管理群群主管**書面同意

## 拾壹、電腦資訊檔案管制（SR-017）⭐

### 檔案管制等級

| 等級 | 說明 |
|------|------|
| **特殊**等級 | 含個人資料之檔案 |
| **重要**等級 | 不含個人資料之檔案 |
| **限制**等級 | 內部限閱 |
| **一般**等級 | 內部使用 |

- 正式環境：資料庫、資料檔案以**特殊等級**列管
- 正式環境：系統程式、應用程式以**重要等級**列管

## 拾貳、端末電腦設備管理（SR-016）⭐

- 端末設備硬軟體之請購、配置、安裝及維護由**資訊作業服務部**統籌
- 一律**加入網域**並安裝電腦端末設備資產管理軟體
- 本機最高權限一律**收回**；停用本機預設**訪客帳號（Guest）**
- 使用USB等外接裝置需經**資訊作業服務部科長級以上**核准
- 筆記型電腦需安裝：**個人防火牆、入侵偵測、防毒軟體、間諜程式移除軟體**
- 筆記型電腦儲放客戶帳務/營運資料→須**壓縮+保護密碼+硬碟加密+身分認證**
- **禁止**架設無線網路基地台，禁止連結外部基地台
- 非本行電腦**不得連接**內部網路

## 拾肆、高頻考點速記

```
【電腦系統三類】
- 第一類：直接客戶自動化服務/重大影響（ATM/網銀/SWIFT/分行）
- 第二類：人工介入間接服務（客服/作業中心）
- 第三類：無客戶接觸/無營運影響（人資/財會/IoT）

【營運持續關鍵名詞】MTPD/RTO/RPO/BIA
- BIA：每半年一次，由資作部執行

【密碼】12字元/90天/前3代不重複/錯誤5次鎖定/最短使用5天
【物聯網密碼】至少6位

【機房】封閉式/錄影3個月/溫度16~25°C/濕度40~70%
【滅火】對設備損害最小（避免灑水式）

【最高權限】A/B part密碼/保險櫃/錄影5年/每日抽檢10件
【正式環境使用時間】下午1時起

【變更等級】一般 vs 重要
【農曆春節】前一個月起禁止變更（除例外）

【緊急通報】30分鐘內/召集人=事故發生部門部主管

【金融卡廢卡】2年或2,000張→副總核准→駐行稽核會同銷毀

【硬碟銷毀】消磁→物理破壞→合格廠商實體銷毀
【格紋袋Y-003/密封袋Y-005】

【LD01→副主管決行 / LD02→線上簽核】

【SR-003核定：總經理 / SR-037核定：資訊管理群群主管】
```
